
# Go Tutorials

Its a series of short Go tutorials to help me learn the language step-by-step.

## Tutorial 1: Basics
- **Location:** `cmd/1.Basics/main.go`
- **Overview:** Introduces Go basics: variables, types, constants, and more.

### Run Tutorial 1
```bash
go run cmd/1.Basics/main.go
```
